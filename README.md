## Português

### 📋 Sobre o Projeto

Landing page moderna e responsiva para a **Barbearia Navalha**, desenvolvida com as mais modernas práticas de desenvolvimento web. O projeto apresenta um design elegante com tema preto e dourado, otimizado para desktop e mobile.

### ✨ Features

- ✅ Design responsivo (mobile-first)
- ✅ Navegação suave com scroll automático
- ✅ Seção de serviços com preços
- ✅ Integração com WhatsApp
- ✅ Performance otimizada com Vite
- ✅ TypeScript para type safety

### 🛠️ Tecnologias

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool ultra-rápido
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Ícones modernos
- **React Icons** - Ícones adicionais (WhatsApp, Instagram)

### 📦 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/landing-page-barbearia.git
cd landing-page-barbearia
```

2. Instale as dependências:
```bash
npm install
```

### 🚀 Como Usar

**Desenvolvimento:**
```bash
npm run dev
```
A aplicação estará disponível em `http://localhost:5174`

**Build para produção:**
```bash
npm run build
```

**Preview da build de produção:**
```bash
npm run preview
```

**Lint:**
```bash
npm run lint
```

### 📁 Estrutura do Projeto

```
src/
├── components/
│   ├── shared/
│   │   ├── Header.tsx      # Navegação fixa com menu responsivo
│   │   ├── Footer.tsx      # Rodapé com links sociais
│   │   ├── Button.tsx      # Componente de botão reutilizável
│   │   └── WhatsAppButton.tsx # Botão flutuante WhatsApp
│   ├── Hero.tsx            # Seção principal com CTA
│   ├── Services.tsx        # Listagem de serviços
│   ├── ServiceCard.tsx     # Card individual de serviço
│   └── Contact.tsx         # Seção de contato
├── assets/                 # Imagens e recursos
├── styles/
│   └── globals.css        # Estilos globais
├── App.tsx                # Componente principal
└── main.tsx               # Entrada da aplicação
```
